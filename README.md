# coffe-sales-dashboard-
#Developed an interactive Coffee Sales Dashboard in Excel to analyze sales performance and customer behavior.

# KEY FEATURES
●	 Developed an interactive Coffee Sales Dashboard in Excel to analyze sales performance and customer behavior  
●	 Integrated multiple datasets (Orders, Customers, Products) using XLOOKUP to create a unified    dataset  
●	Cleaned and transformed raw data to ensure accuracy and consistency for analysis  
●	Created pivot tables and charts to analyze total sales trends, top customers, and country-wise performance  
●	Designed dynamic dashboards with slicers to filter data by product category, size, and time period  
●	Identified high-performing products and key customer segments to support business decision-making  
●	Analyzed 5,000+ sales records to uncover trends and improve data-driven insights
